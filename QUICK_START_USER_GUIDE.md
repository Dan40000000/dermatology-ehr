# Quick Start Guide - Dermatology EHR

**Get up and running in 5 minutes**

---

## 5-Minute Tour

### 1. Login (30 seconds)

1. Open browser → Navigate to your EHR URL
2. Enter email, password, and tenant ID
3. Click **Login**

**First time?** You'll be prompted to change password.

---

### 2. Dashboard Overview (1 minute)

Your **Home** screen shows:

**Top Cards:**
- Today's appointments
- Patients seen
- Revenue
- Pending tasks

**Quick Actions:**
- New Patient
- Schedule Appointment
- Create Note
- Send Message

**Navigation:** Left sidebar has all sections

---

### 3. Find a Patient (30 seconds)

1. Click **Patients** in sidebar
2. Type in search box:
   - Patient name
   - Date of birth
   - Phone number
   - Medical record number
3. Click patient to open chart

---

### 4. Schedule an Appointment (1 minute)

1. Click **Schedule** in sidebar
2. Click empty time slot
3. Fill in:
   - Patient (search or create new)
   - Appointment Type
   - Provider
   - Chief Complaint
4. Click **Save**

**That's it!** Patient receives confirmation.

---

### 5. Document a Visit (2 minutes)

1. From patient chart → **+ New Encounter**
2. Use template or start blank
3. Fill SOAP sections:
   - **S:** Chief complaint + HPI
   - **O:** Vitals + exam findings
   - **A:** Diagnosis (search ICD-10)
   - **P:** Treatment plan
4. Add orders (prescriptions, labs)
5. Click **Sign Note**

**Faster?** Use voice-to-text or Ambient AI Scribe!

## Common Workflows

### Check-In a Patient

**Office Flow → Select patient → Mark "Checked In"**

Patient status updates automatically on schedule.

---

### Take Vitals

**Open patient chart → Vitals tab → Enter BP, HR, Temp, Weight → Save**

Auto-calculates BMI.

---

### E-Prescribe

**In encounter → Add Prescription → Search medication → Fill details → Select pharmacy → Send**

Prescription sent electronically in seconds.

---

### Order a Lab

**In encounter → Add Order → Select Lab → Choose tests → Link diagnosis → Submit**

Order sent to lab electronically.

---

### Review Lab Results

**Labs page → Click result → Review → Sign → Notify patient**

Results auto-import from lab.

---

### Send a Text Message

**Text Messages → Select patient → Type message → Send**

Use templates for quick responses.

---

### Create a Bill

**After signing note → Go to Financials → Claims tab → Select claim → Submit**

Claim sent to insurance via clearinghouse.

---

### Generate a Report

**Reports → Select report type → Set date range → Run Report → Export CSV/PDF**

## Tips and Tricks

### 1. Use Templates

**Save time on common visits**

- Navigate to encounter → Click **Use Template**
- Select "Acne Visit", "Skin Check", etc.
- Auto-fills common sections
- Customize as needed

**Create your own:**
- Settings → Note Templates → + New Template

---

### 2. Quick Patient Search

**Type anywhere to search**

- Press **Ctrl+K**
- Type patient name
- Hit Enter
- Patient chart opens

**Works with:**
- Patient names
- MRN
- Phone numbers
- DOB

---

### 3. Duplicate Appointments

**Quickly schedule recurring visits**

- Right-click existing appointment
- Click **Duplicate**
- Change date/time
- Save

Perfect for regular follow-ups!

---

### 4. Batch Operations

**Process multiple items at once**

**Claims:**
- Select multiple claims (checkboxes)
- Click **Submit Selected**
- All submitted together

**Notes:**
- Select multiple notes
- **Finalize Selected** or **Assign Selected**

**Prescriptions:**
- Select multiple Rx
- **ePrescribe Selected**

---

### 5. Favorite Patients

**Quick access to frequent patients**

- In patient chart → Click star icon
- Patient added to favorites
- Access from dashboard Quick Links

---

### 6. Smart Filters

**Find exactly what you need**

Most pages have filters:
- **Date range**
- **Provider**
- **Status**
- **Patient**

**Save filter sets:**
- Configure filters → **Save View**
- Name it (e.g., "My Unsigned Notes")
- Quick access from dropdown

---

### 7. Export Everything

**Data portability**

Most tables have export buttons:
- **CSV:** For Excel
- **PDF:** For printing
- **Print:** Direct to printer

Great for:
- Sharing with accountant
- Monthly reviews
- Presentations

---

### 8. Use Macros

**Type less, document more**

Create text shortcuts:
- `.normal` → "Physical examination normal"
- `.fu2w` → "Follow-up in 2 weeks"
- `.acne` → Full acne documentation template

**Set up in:** Settings → Macros

---

### 9. Multi-Tab Workflow

**Work efficiently**

Open multiple tabs/windows:
- Tab 1: Patient chart
- Tab 2: Schedule
- Tab 3: Current encounter

**Right-click** links → **Open in New Tab**

---

### 10. Mobile Access

**On the go**

- Fully responsive design
- Works on tablets and phones
- Perfect for:
  - Checking schedule
  - Reviewing lab results
  - Sending messages
  - Viewing patient charts

**Tip:** Add to home screen for app-like experience

## Role-Specific Quick Starts

### For Front Desk

**Daily workflow:**

1. **Check Today's Schedule**
   - Schedule page → Today view
   - See all appointments

2. **Check In Patients**
   - Office Flow → Select patient → Check In
   - Collect copay
   - Update insurance if needed

3. **Answer Phone**
   - Schedule appointments
   - Take messages (Tasks module)
   - Verify insurance

4. **End of Day**
   - Review no-shows
   - Confirm tomorrow's appointments (text reminders)

---

### For Medical Assistants

**Daily workflow:**

1. **Prepare Rooms**
   - Office Flow → See patient queue

2. **Room Patients**
   - Office Flow → Mark "In Room"
   - Take vitals
   - Update allergies/medications

3. **Clinical Support**
   - Assist with procedures
   - Take clinical photos
   - Label specimens

4. **Tasks**
   - Follow up on lab results
   - Call patients
   - Schedule appointments

---

### For Providers

**Daily workflow:**

1. **Review Schedule**
   - Check today's appointments
   - Review patient charts before visits

2. **See Patients**
   - Open encounter from schedule
   - Use templates or Ambient AI
   - Document SOAP note
   - Order prescriptions and labs
   - Sign note

3. **Review Results**
   - Labs page → Sign off results
   - Notify patients

4. **Messages**
   - Respond to patient portal messages
   - Review and approve refill requests

---

### For Billing Staff

**Daily workflow:**

1. **Scrub Claims**
   - Financials → Claims → Draft
   - Fix errors (missing diagnoses, etc.)

2. **Submit Claims**
   - Select ready claims
   - Submit to clearinghouse

3. **Post Payments**
   - Review ERAs
   - Post payments to claims
   - Reconcile with EFTs

4. **Follow Up**
   - Denied claims → Fix and resubmit
   - Aging report → Collections calls

## Common Questions

### How do I reset my password?

**Forgot password:** Login page → Forgot Password → Enter email → Check email for reset link

**Change password:** Click your name (top right) → Profile → Change Password

---

### Can I undo a signed note?

**No.** Signed notes are locked for legal/compliance reasons.

**Solution:** Add an addendum
- Open signed note → **Add Addendum** → Enter correction

---

### How do I add a new user?

**Admin only:** Settings → Users → + New User → Enter details → Save

---

### Where are deleted items?

Most deletions are "soft deletes" (hidden, not removed).

**To restore:** Contact administrator or check Audit Log.

---

### How do I print?

**Any page:** Use browser print (Ctrl+P or Cmd+P)

**Better:** Use export buttons → PDF → Print PDF

---

### Can I customize the system?

**Yes!**
- Note templates
- Appointment types
- Fee schedules
- Report configurations
- Message templates
- Macros

**Most customization in:** Settings section

---

### Is there a mobile app?

**Web-based:** No separate app needed, works in any mobile browser.

**Add to home screen:**
- iPhone: Safari → Share → Add to Home Screen
- Android: Chrome → Menu → Add to Home Screen

Looks and acts like native app!
